# ResumeIQ

### AI-Powered Resume Review Assistant using Anthropic Claude

ResumeIQ is an AI-powered web application that analyzes a resume against a job description and generates recruiter-style feedback using the Anthropic Claude API.

The application helps job seekers understand how well their resume matches a specific role by providing actionable insights, identifying missing skills, and suggesting improvements.

---

## Features

- Compare a resume against a job description
- Generate an AI-powered Resume Match Score
- Identify key strengths
- Detect missing skills
- Recommend resume improvements
- Provide a recruiter-style hiring recommendation
- Simple and interactive Streamlit web interface

## How It Works

1. Paste your resume into the Resume section.
2. Paste a job description into the Job Description section.
3. Click **Analyze Resume**.
4. Claude evaluates the resume and generates:
   - Resume Match Score
   - Top Strengths
   - Missing Skills
   - Resume Improvement Suggestions
   - Final Hiring Recommendation
